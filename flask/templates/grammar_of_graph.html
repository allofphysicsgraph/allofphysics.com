Grammar of the controlled natural language

In the Physics Derivation Graph, the primary vocabulary is a set of inference rules that relate mathematical expressions.
The grammar of the Physics Derivation Graph is specified by this list, with all items joined by "and":

  • The Physics Derivation Graph is a set of derivations.
  • Each derivation has one or more steps.
  • Each step has one inference rule.
  • Each inference rule has at least one expression.
  • Each inference rule has zero or more input expressions.
  • Each inference rule has zero or more output expressions.
  • Some inference rules have one or more required feeds.
  • Each expression and each feed is composed of symbols.
  • Each expression and each feed is represented by an Abstract Syntax Tree.
  • Each expression contains one or more relation operators (e.g., =, <, >, ≠).
  • A symbol is either an operator or a constant or a variable.
  • Every symbol is uniquely identified by a numeric value.
  • Every operator acts on one or more symbols.
  • Every constant and variable is dimensionless or has dimenions.
  • Dimenions are from the set of seven SI base quantities: time, length, mass, electric current, thermodynamic temperature, amount of substance, luminous intensity.
  • Constants have a numeric value and are either dimensionless or have a unit.
  • Every unit is associated with one or more dimenions.

The full list of vocabulary used in the Physics Derivation Graph is documented in the list of inference rules and list of expressions and list of symbols. {% endblock %}

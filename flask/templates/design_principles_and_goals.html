This page enumerates design principles and goals for the Physics Derivation Graph (PDG).
The list is unordered.

  • Technology used should be free to improve accessibility.
  • Technology used should be open source to improve ability to inspect the implementation.
  • Technology used should be durable and stable to decrease the maintenance burden.
  • Rendered HTML pages should have a single scope. (As opposed to a single-page website.)
  • HTML tag "Fieldset" is used to denote actions available to the user.
